AllTheFeels

Amazon's Alexa script prompts the user how his/her day is going, then depending on the sentiment expressed by the user, it produces a list of regularly-updated Spotify playlists tailored to his/her mood. Worked in incorporating the AlchemyAPI to the response from Alexa, then sorting a list of playlist in accordance to the sentiment in the person's day. Check the Devpost here!

SummarizeMe

File receives an article (or any body of text) and produces a summary around what sentences have the most value. Reduces big bodies of text to a 70-80% of it's original length. Written in Python using the library NLTK. Project inspired by the auto TL;DR bot used in some subreddits.
